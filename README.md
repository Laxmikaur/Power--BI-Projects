# 🛒 Flipkart Sales Dashboard (Power BI)

## 📌 Overview

This project showcases an interactive sales dashboard built using Power BI to analyze Flipkart data. It helps in understanding sales performance, customer behavior, and product trends through clear and dynamic visualizations.

---

## 🚀 Features

* Total Revenue, Total Orders, and Average Customer Rating
* Year-wise trend analysis of revenue and orders
* Product Category-wise sales performance
* Order distribution by delivery type (Standard, Express, etc.)
* Customer segmentation based on gender
* Interactive filters for better data exploration

---

## 📈 Key Insights

* Health & Beauty and Fashion categories are top contributors
* Revenue shows strong growth in recent years
* Orders have increased significantly over time
* Customer distribution is balanced across genders
